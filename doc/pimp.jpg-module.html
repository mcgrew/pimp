read(filename)

source code 
Reads a jpeg file.
Parameters:
  • filename (string) - the name of the file to be read.
Returns: tuple
A tuple ( width, height, data ). Width and height are in pixels, data is a string containing chr(red) + chr(green) + chr(blue) for each pixel.

Writes data out to a jpeg file
Parameters:
  • filename (string) - The name of the file to write to.
  • width (int) - The width of the image in pixels
  • height (int) - The height of the image in pixels
  • data (string) - The data as a string.
Returns: boolean
True on success
