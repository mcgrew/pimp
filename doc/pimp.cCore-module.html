spatial(...)

 
Implements an arbitrary spatial filter on the image. The filter should be a tuple containing integers. The number of elements should be the square of an odd number. The filter will be truncated to the nearest square of an odd number if this is not the case.
Parameters:
  • width (int) - The width of the image in pixels
  • height (int) - The height of the image in pixels
  • data (string) - The data as a string - chr(red) + chr(green) + chr(blue) for each pixel.
  • filter (tuple) - A 9 element tuple containing the filter values.
  • filterTotal (int) - The total weight of the filter the final result is to be divided by. This is optional and automatically determined if left out.
Returns: tuple
A tuple ( width, height, data ). Width and height are in pixels, data is a string containing chr(red) + chr(green) + chr(blue) for each pixel.

swapChannels(...)

 

Swaps one channel with another in an image. The channels are numbered as follows: For a 3 or 4 channel image (RGB or RGBA): 0. Red 1. Green 2. Blue 3. Alpha

For a 1 or 2 channel image 0. Grey 1. Alpha

Parameters:
  • width (int) - The width of the image in pixels
  • height (int) - The height of the image in pixels
  • data (string) - The data as a binary string.
  • channel1 (int) - The first of the 2 channels to swap
  • channel2 (int) - The second of the 2 channels to swap
Returns: string
The new data as a string.

table(...)

 
Replaces the data in an image based on a substitution table. The table should contain 256 values, each containing the value to be substituted. It is treated as a lookup table; a value of 205 will be replaced with whatever is at index 205 in the table, and so forth.
Parameters:
  • width (int) - The width of the image in pixels
  • height (int) - The height of the image in pixels
  • data (string) - The data as a binary string.
  • sub_table (tuple) - A tuple containg 256 integer values used for substitution.
Returns: tuple
A tuple ( width, height, data ). Width and height are in pixels, data is a string containing the binary data.
