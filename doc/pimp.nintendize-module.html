Function Details [hide private]

execute(...)

 
Pixellates the image and reduces bit quality in a way such that it looks like an image from a Nintendo Game. Yes, this was just for fun.
Parameters:
  • filename (string) - The name of the file to write to.
  • width (int) - The width of the image in pixels
  • height (int) - The height of the image in pixels
  • data (string) - The data as a string - chr(red) + chr(green) + chr(blue) for each pixel.
Returns: tuple
A tuple ( width, height, data ). Width and height are in pixels, data is a binary string.
